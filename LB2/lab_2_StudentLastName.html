    1. Що таке pip і conda та чим вони відрізняються між собою?

      pip — це стандартний менеджер пакетів для Python, який використовується для встановлення бібліотек з офіційного репозиторію PyPI.

      conda — це потужніший інструмент, який не лише встановлює пакети, а й керує середовищами. Він входить до складу дистрибутива Anaconda і дозволяє встановлювати як Python-бібліотеки, так і пакети, написані на інших мовах (наприклад, C або R).

      Відмінність: pip працює лише з PyPI і не керує середовищами, тоді як conda має власний репозиторій і дозволяє створювати та ізолювати середовища.

    2. Як виглядатиме виконання пунктів 1–3 індивідуального завдання за допомогою Virtualenv?

      1. Створення віртуального середовища:

      python -m venv myenv

      2. Активація середовища:

      Windows:

      myenv\Scripts\activate

      Linux/macOS:

      source myenv/bin/activate

      3. Встановлення необхідних бібліотек:

      pip install opencv-python numpy matplotlib

    4. Процедура сортування масиву методом бульбашки:

      Метод бульбашки — це простий алгоритм сортування, який порівнює сусідні елементи і обмінює їх, якщо вони у неправильному порядку.

      def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr
